## 一、总览与动机

### 1.1 当前技术债务

| 维度 | 当前方案 | 问题 |
|------|----------|------|
| 音频编码 | 无（PCM 128kbps） | 带宽浪费 8.4 倍 |
| 视频编码 | JPEG (~1.2Mbps) | H.264 可压缩到 ~200kbps |
| 传输协议 | TCP 全量 struct | TCP 拥塞控制不适合实时媒体 |
| 视频渲染 | QLabel QPixmap (CPU) | 浪费 GPU 硬件加速能力 |
| 音频播放 | QAudioOutput | 延迟较高，SDL2 可更精细控制 |
| 视频采集 | QCamera + QVideoProbe | 灵活性不足，YUV→RGB 手动转换 |

### 1.2 升级目标

```
当前（Phase 1）                      目标（Phase 2）
─────────────────────────────       ───────────────────────────
PCM 8kHz 16-bit 128kbps      →     Opus 8kHz 16-32kbps
JPEG Q70 320×240 ~1.2Mbps    →     H.264 320×240 ~200kbps
TCP 定长帧 struct              →     信令 TCP + 媒体 UDP/RTP
QLabel QPixmap (CPU)          →     OpenGL Texture (GPU)
QAudioOutput                  →     SDL2 低延迟播放
QCamera                       →     OpenCV VideoCapture
无                             →     CNN 人脸检测
无                             →     MP4 推流录制
```

## 二、当前状态与目标对比

### 2.1 音频链路对比

```
当前：
麦克风 → QAudioInput(8kHz) → PCM 640B/40ms → TCP → QAudioOutput → 扬声器
带宽: 128kbps | 延迟: ~80ms | 编码: 无

目标：
麦克风 → QAudioInput(8kHz) → Opus编码(76B/40ms) → UDP/RTP → Opus解码 → SDL2 → 扬声器
带宽: ~16kbps | 延迟: ~40ms | 编码: Opus (8.4:1 压缩)
```

### 2.2 视频链路对比

```
当前：
摄像头 → QCamera(YUYV 1280×720) → 手动BT.601 RGB → QImage scaled 320×240
→ JPEG Q70 (5-20KB) → TCP → QPixmap JPEG解码(CPU) → QLabel
带宽: ~1.2Mbps@15fps | 延迟: ~200ms | 编码: JPEG

目标：
摄像头 → OpenCV VideoCapture(BGR) → cv::cvtColor(BGR→RGB) → cv::resize(320×240)
→ FFmpeg H.264编码 → UDP/RTP → FFmpeg H.264解码 → OpenGL纹理(GPU) → QOpenGLWidget
带宽: ~200kbps@15fps | 延迟: ~100ms | 编码: H.264 (102:1 压缩比)
```

### 2.3 数据量计算对比

| 场景 | 当前 (PCM+JPEG/TCP) | 目标 (Opus+H.264/UDP) | 节省 |
|------|---------------------|------------------------|------|
| 仅音频 | 128 kbps (16 KB/s) | ~16 kbps (2 KB/s) | 87% |
| 仅视频 | ~1.2 Mbps (150 KB/s) | ~200 kbps (25 KB/s) | 83% |
| 音视频混合 | ~1.33 Mbps (166 KB/s) | ~216 kbps (27 KB/s) | 84% |

### 3.2 选型决策

#### 音频编解码：Opus 1.4（首选）vs FFmpeg AAC

| 维度 | Opus | AAC (FFmpeg) | 决策 |
|------|------|-------------|------|
| 压缩率 | 8-12:1 | 8-10:1 | ≈ |
| 延迟 | 低（5ms帧）| 中（20ms帧）| Opus 胜 |
| 复杂度 | 低（单一库） | 高（需FFmpeg全套） | Opus 胜 |
| 库大小 | 648KB (.a) | ~5MB (avcodec) | Opus 胜 |
| 已有资料 | 有编译好的静态库+文档 | 有源码+PDF教程 | Opus 胜 |

**决策：使用 Opus 1.4。** 已有编译好的 `libopus.a` + PDF 编译文档，低延迟，小体积。

#### 视频编解码：FFmpeg H.264

唯一选择。材料库有 ffmpeg-4.3.1 完整源码 + ffmpeg-4.2.2 预编译（32位）。需重编译为64位。

#### 视频渲染：OpenGL (QOpenGLWidget)

材料库已有完整模板 `MyOpenGLWidget`，直接使用。

#### 音频播放：SDL2

材料库有 SDL2-2.0.10 预编译 64 位 DLL。用于替代 QAudioOutput 获得更低延迟。

#### 视频采集：OpenCV VideoCapture

材料库有 OpenCV 4.5.4 + contrib（含 face 模块）。替代 QCamera 获得更灵活的图像处理。

```cpp
// audio/opus_encoder.cpp
#include "opus_encoder.h"
#include <QDebug>

OpusEncoder::OpusEncoder() : m_enc(nullptr), m_sampleRate(8000), m_channels(1), m_initialized(false) {}

OpusEncoder::~OpusEncoder() {
    if (m_enc) opus_encoder_destroy(m_enc);
}

bool OpusEncoder::init(int sampleRate, int channels) {
    m_sampleRate = sampleRate;
    m_channels = channels;
    int err = 0;
    m_enc = opus_encoder_create(sampleRate, channels, OPUS_APPLICATION_VOIP, &err);
    if (err != OPUS_OK) {
        qDebug() << __func__ << "Opus encoder create failed:" << err;
        return false;
    }
    // 设置为 16kbps 窄带语音
    opus_encoder_ctl(m_enc, OPUS_SET_BITRATE(16000));
    // 信号处理：语音优化
    opus_encoder_ctl(m_enc, OPUS_SET_SIGNAL(OPUS_SIGNAL_VOICE));
    // 复杂度：平衡质量与速度
    opus_encoder_ctl(m_enc, OPUS_SET_COMPLEXITY(5));
    m_initialized = true;
    qDebug() << __func__ << "Opus encoder initialized:" << sampleRate << "Hz," << channels << "ch";
    return true;
}

QByteArray OpusEncoder::encode(const QByteArray& pcm) {
    if (!m_initialized) return QByteArray();
    // 40ms @ 8kHz mono = 320 samples = 640 bytes PCM
    int frameSize = m_sampleRate / 25;  // 320 samples for 40ms
    int maxEncoded = 4000;  // worst case
    QByteArray encoded(maxEncoded, '\0');
    int nbytes = opus_encode(m_enc,
        reinterpret_cast<const opus_int16*>(pcm.constData()),
        frameSize,
        reinterpret_cast<unsigned char*>(encoded.data()),
        maxEncoded);
    if (nbytes < 0) {
        qDebug() << __func__ << "Opus encode error:" << nbytes;
        return QByteArray();
    }
    encoded.resize(nbytes);
    return encoded;
}

void OpusEncoder::reset() {
    if (m_enc) opus_encoder_ctl(m_enc, OPUS_RESET_STATE);
}
```

### 任务 2.3：创建 Opus 解码器封装


```cpp
// audio/opus_decoder.cpp
#include "opus_decoder.h"
#include <QDebug>

OpusDecoder::OpusDecoder() : m_dec(nullptr), m_sampleRate(8000), m_channels(1), m_initialized(false) {}

OpusDecoder::~OpusDecoder() {
    if (m_dec) opus_decoder_destroy(m_dec);
}

bool OpusDecoder::init(int sampleRate, int channels) {
    int err = 0;
    m_dec = opus_decoder_create(sampleRate, channels, &err);
    if (err != OPUS_OK) {
        qDebug() << __func__ << "Opus decoder create failed:" << err;
        return false;
    }
    m_initialized = true;
    return true;
}

QByteArray OpusDecoder::decode(const QByteArray& encoded) {
    if (!m_initialized || encoded.isEmpty()) return QByteArray();
    int frameSize = m_sampleRate / 25;  // 320 samples
    QByteArray pcm(frameSize * 2, '\0');  // 640 bytes (16-bit)
    int nbytes = opus_decode(m_dec,
        reinterpret_cast<const unsigned char*>(encoded.constData()),
        encoded.size(),
        reinterpret_cast<opus_int16*>(pcm.data()),
        frameSize,
        0);  // 0 = no FEC
    if (nbytes < 0) {
        qDebug() << __func__ << "Opus decode error:" << nbytes;
        return QByteArray();
    }
    pcm.resize(nbytes * 2);
    return pcm;
}
```

### 任务 2.5：修改 slots_sendAudioFrame 使用 Opus

**文件**：修改 `ckernel.cpp` 中 `slots_sendAudioFrame`

```cpp
void CKernel::slots_sendAudioFrame(QByteArray ba) {
    // 旧路径: 直接发送 PCM 640B
    // PROT_AUDIO_FRAME frame; memcpy(frame.data, ba.data(), 640);
    
    // 新路径: Opus 编码后发送
```

```cpp
// audio/sdl_audio_player.cpp
#include "sdl_audio_player.h"
#include <QDebug>

SdlAudioPlayer::SdlAudioPlayer(QObject* parent) : QObject(parent),
    m_device(0), m_sampleRate(8000), m_channels(1), m_initialized(false), m_running(false) {}

SdlAudioPlayer::~SdlAudioPlayer() {
    stop();
    if (m_device) SDL_CloseAudioDevice(m_device);
    SDL_QuitSubSystem(SDL_INIT_AUDIO);
}

bool SdlAudioPlayer::init(int sampleRate, int channels) {
    if (SDL_InitSubSystem(SDL_INIT_AUDIO) < 0) {
        qDebug() << __func__ << "SDL audio init failed:" << SDL_GetError();
        return false;
    }
    SDL_AudioSpec desired, obtained;
    SDL_zero(desired);
    desired.freq = sampleRate;
    desired.format = AUDIO_S16LSB;  // 16-bit signed, little-endian
    desired.channels = channels;
    desired.samples = 320;  // 40ms @ 8kHz
    desired.callback = audioCallback;
    desired.userdata = this;

    m_device = SDL_OpenAudioDevice(nullptr, 0, &desired, &obtained, 0);
    if (!m_device) {
        qDebug() << __func__ << "SDL open audio device failed:" << SDL_GetError();
        return false;
    }
    m_sampleRate = obtained.freq;
    m_channels = obtained.channels;
    m_initialized = true;
    qDebug() << __func__ << "SDL audio initialized:" << m_sampleRate << "Hz," << m_channels << "ch";
    return true;
}

void SdlAudioPlayer::start() {
    if (!m_initialized || m_running) return;
    SDL_PauseAudioDevice(m_device, 0);
    m_running = true;
}

void SdlAudioPlayer::stop() {
    if (!m_running) return;
    SDL_PauseAudioDevice(m_device, 1);
    m_running = false;
}

void SdlAudioPlayer::slot_playAudio(const QByteArray& pcm) {
    QMutexLocker locker(&m_mutex);
    if (m_bufferQueue.size() > 10) {
        // 丢弃旧帧防止延迟累积
        m_bufferQueue.dequeue();
    }
    m_bufferQueue.enqueue(pcm);
}

void SdlAudioPlayer::audioCallback(void* userdata, Uint8* stream, int len) {
    SdlAudioPlayer* player = static_cast<SdlAudioPlayer*>(userdata);
    QMutexLocker locker(&player->m_mutex);
    
    if (player->m_bufferQueue.isEmpty()) {
        SDL_memset(stream, 0, len);  // 静音
        return;
    }
    
    QByteArray pcm = player->m_bufferQueue.dequeue();
    int copyLen = qMin(pcm.size(), len);
    SDL_memcpy(stream, pcm.constData(), copyLen);
    if (copyLen < len) {
        SDL_memset(stream + copyLen, 0, len - copyLen);
    }
}
```

```cpp
#include "h264_encoder.h"
#include <QDebug>

H264Encoder::H264Encoder(QObject* parent) : QObject(parent),
    m_ctx(nullptr), m_frame(nullptr), m_pkt(nullptr), m_swsCtx(nullptr),
    m_width(320), m_height(240), m_fps(15), m_pts(0), m_initialized(false) {}

H264Encoder::~H264Encoder() {
    if (m_swsCtx) sws_freeContext(m_swsCtx);
    if (m_frame) av_frame_free(&m_frame);
    if (m_pkt) av_packet_free(&m_pkt);
    if (m_ctx) {
        avcodec_free_context(&m_ctx);
    }
}

bool H264Encoder::init(int width, int height, int fps, int bitrate) {
    m_width = width;
    m_height = height;
    m_fps = fps;

    // 查找 H.264 编码器
    const AVCodec* codec = avcodec_find_encoder(AV_CODEC_ID_H264);
    if (!codec) {
        qDebug() << __func__ << "H.264 encoder not found";
        return false;
    }

    m_ctx = avcodec_alloc_context3(codec);
    m_ctx->width = width;
    m_ctx->height = height;
    m_ctx->time_base = AVRational{1, fps};
    m_ctx->framerate = AVRational{fps, 1};
    m_ctx->pix_fmt = AV_PIX_FMT_YUV420P;
    m_ctx->bit_rate = bitrate;
    m_ctx->gop_size = fps * 2;  // 2秒关键帧间隔
    m_ctx->max_b_frames = 0;  // 低延迟：无B帧
    // 低延迟预设
    av_opt_set(m_ctx->priv_data, "preset", "ultrafast", 0);
    av_opt_set(m_ctx->priv_data, "tune", "zerolatency", 0);

    if (avcodec_open2(m_ctx, codec, nullptr) < 0) {
        qDebug() << __func__ << "Cannot open H.264 encoder";
        return false;
    }

    m_frame = av_frame_alloc();
    m_frame->format = m_ctx->pix_fmt;
    m_frame->width = width;
    m_frame->height = height;
    av_frame_get_buffer(m_frame, 0);

    m_pkt = av_packet_alloc();

    // RGB→YUV420P 转换器
    m_swsCtx = sws_getContext(width, height, AV_PIX_FMT_BGRA,
        width, height, AV_PIX_FMT_YUV420P,
        SWS_BILINEAR, nullptr, nullptr, nullptr);

    m_initialized = true;
    m_pts = 0;
    qDebug() << __func__ << "H.264 encoder initialized:" << width << "x" << height
             << "@" << fps << "fps," << bitrate/1000 << "kbps";
    return true;
}

QByteArray H264Encoder::encode(const QImage& rgbImage) {
    if (!m_initialized) return QByteArray();

    // QImage → BGRA → AVFrame (YUV420P)
    QImage bgra = rgbImage.convertToFormat(QImage::Format_RGBA8888);
    const uint8_t* srcSlice[1] = { bgra.constBits() };
    int srcStride[1] = { static_cast<int>(bgra.bytesPerLine()) };
    sws_scale(m_swsCtx, srcSlice, srcStride, 0, m_height,
              m_frame->data, m_frame->linesize);

    m_frame->pts = m_pts++;

    // 编码
    int ret = avcodec_send_frame(m_ctx, m_frame);
    if (ret < 0) return QByteArray();

    QByteArray result;
    while (ret >= 0) {
        ret = avcodec_receive_packet(m_ctx, m_pkt);
        if (ret == AVERROR(EAGAIN) || ret == AVERROR_EOF) break;
        if (ret < 0) return QByteArray();
        result.append((const char*)m_pkt->data, m_pkt->size);
        av_packet_unref(m_pkt);
    }
    return result;
}

void H264Encoder::flush() {
    if (!m_initialized) return;
    avcodec_send_frame(m_ctx, nullptr);
    // 读取剩余数据需要通过回调返回，此处简化
}
```

```cpp
    if (m_frame) av_frame_free(&m_frame);
    if (m_pkt) av_packet_free(&m_pkt);
    if (m_ctx) avcodec_free_context(&m_ctx);

bool H264Decoder::init() {
    const AVCodec* codec = avcodec_find_decoder(AV_CODEC_ID_H264);
    if (!codec) {
        qDebug() << __func__ << "H.264 decoder not found";
        return false;
    }
    m_ctx = avcodec_alloc_context3(codec);
    if (avcodec_open2(m_ctx, codec, nullptr) < 0) {
        qDebug() << __func__ << "Cannot open H.264 decoder";
        return false;
    }
    m_frame = av_frame_alloc();
    m_pkt = av_packet_alloc();

QImage H264Decoder::decode(const QByteArray& h264Nalu) {
    if (!m_initialized || h264Nalu.isEmpty()) return QImage();

    m_pkt->data = (uint8_t*)h264Nalu.constData();
    m_pkt->size = h264Nalu.size();

    int ret = avcodec_send_packet(m_ctx, m_pkt);
    if (ret < 0) return QImage();

    ret = avcodec_receive_frame(m_ctx, m_frame);
    if (ret < 0) return QImage();

    // YUV420P → BGRA (给 OpenGL 纹理)
    if (!m_swsCtx || m_swsCtx->srcW != m_frame->width) {
        if (m_swsCtx) sws_freeContext(m_swsCtx);
        m_swsCtx = sws_getContext(m_frame->width, m_frame->height, AV_PIX_FMT_YUV420P,
            m_frame->width, m_frame->height, AV_PIX_FMT_BGRA,
            SWS_BILINEAR, nullptr, nullptr, nullptr);
    }

    QImage img(m_frame->width, m_frame->height, QImage::Format_RGBA8888);
    uint8_t* dstSlice[1] = { img.bits() };
    int dstStride[1] = { static_cast<int>(img.bytesPerLine()) };
    sws_scale(m_swsCtx, m_frame->data, m_frame->linesize, 0, m_frame->height,
              dstSlice, dstStride);

    av_frame_unref(m_frame);
    return img;
}
```

## 十七、风险与应对

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| FFmpeg 64 位编译失败 | 中 | 高 | 降级方案：OpenCV 内建 FFmpeg |
| Opus 静态库不兼容 64 位 | 中 | 中 | 从 Opus 源码重编译（./configure 配合 x86_64-w64-mingw32） |
| OpenCV 4.5.4 contrib 64 位编译失败 | 中 | 高 | 用 OpenCV 4.5.4 核心模块（不含 contrib），人脸检测用 libfacedetection |
| UDP 丢包严重 | 低 | 中 | 增加 FEC 前向纠错，或降级回 TCP |
| libfacedetection 64 位编译失败 | 低 | 低 | 跳过 M8，人脸检测非核心功能 |
| H.264 编码延迟过高 | 中 | 中 | 使用 `ultrafast` + `zerolatency` 预设，降低分辨率 |
